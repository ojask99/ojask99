## 🚀 Projects

### 🎙️ **Shortscut**
AI-powered podcast clipping tool with sub-2-minute processing pipeline for viral moment detection.  
Built with scalable async architecture using Inngest job queues and Modal for production ML inference.

[![GitHub](https://img.shields.io/badge/GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/ojask99/shortscut)

### 🏥 **HealthAware**
AI medical diagnostics platform with ML models for TB, kidney stone, and COVID-19 diagnosis.  
Smart India Hackathon 2024 finalist with Next.js frontend and Flask APIs deployed on GCP.

[![GitHub](https://img.shields.io/badge/GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/ojask99/healthaware)

Open source project under ACM-VIT with TypeScript and React implementation.
